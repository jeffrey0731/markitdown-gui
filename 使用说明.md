# MarkITDown GUI — 使用说明

## 一、软件简介

MarkITDown GUI 是微软开源工具 **MarkITDown** 的图形化前端，可以将多种文件格式转换为 Markdown 文本。

- 开发语言：Python 3
- 界面框架：tkinter（Python 内置，无需额外安装）
- 依赖库：MarkITDown（`pip install markitdown`）
- 适用系统：Windows 10 / 11

---

## 二、安装与启动

### 1. 安装依赖

打开命令提示符（CMD）或 PowerShell，执行：

```
pip install markitdown
```

如果已安装过但版本较旧，可先升级：

```
pip install --upgrade markitdown
```

### 2. 启动程序

在程序所在目录打开命令提示符，执行：

```
python markitdown_gui.py
```

### 各部分功能

| 区域 | 说明 |
|------|------|
| **转换类别** | 下拉选择要转换的文件类型，切换后右侧会显示该类别的支持格式 |
| **文件路径** | 显示已选择的文件路径；也可手动输入 |
| **浏览…** | 打开文件选择对话框，直接选取要转换的文件 |
| **粘贴链接** | 弹窗输入在线 URL（如 Wikipedia 页面、YouTube 链接等） |
| **开始转换** | 执行转换，结果显示在下方的文本区域 |
| **清空结果** | 清除文本区域内容和已保存状态 |
| **保存 .md** | 将转换结果保存为 Markdown 文件（.md）或纯文本文件（.txt） |
| **复制** | 一键将转换结果复制到剪贴板 |
| **状态栏** | 底部显示当前操作状态提示 |

## 四、支持的转换类别

| 类别 | 支持格式 | 说明 |
|------|---------|------|
| 📄 文档 | PDF、Word(.docx)、EPUB | 提取正文文本 |
| 📊 表格 | Excel(.xlsx)、CSV | 转为 Markdown 表格 |
| 📽 演示 | PowerPoint(.pptx) | 提取幻灯片文字 |
| 💻 代码 | Jupyter Notebook(.ipynb) 及常见代码文件 | 保留代码块格式 |
| 🌐 网页 | HTML、HTM、XML、RSS | 提取网页正文 |
| 🔗 在线链接 | URL 地址 | Wikipedia、YouTube 等在线内容 |
| 🖼 图片 | JPG、PNG、GIF、BMP、WebP、TIFF | AI 生成图片描述（需额外配置） |
| 🎵 音频 | MP3、WAV、FLAC、M4A、OGG、WMA | 语音转文字（需额外配置） |
| 📦 压缩包 | ZIP | 提取压缩包内文件内容 |
| 📧 邮件 | Outlook(.msg) | 提取邮件正文 |
| 📝 纯文本 | TXT、LOG、INI、CFG、CONF 等 | 直接读取文本内容 |

> **提示：** 浏览文件时不过滤格式，所有文件均可选择。程序会根据文件格式自动判断是否可转换。

## 五、操作步骤

### 转换本地文件

1. 在「转换类别」下拉框中选择文件所属类别（可选，不影响实际转换）
2. 点击「浏览…」按钮，在弹出的对话框中选择要转换的文件
3. 点击「开始转换」
4. 等待转换完成，结果显示在下方文本区域
5. 点击「保存 .md」可将结果导出为文件；或点击「复制」粘贴到其他位置

### 转换在线链接

1. 在「转换类别」中选择「🔗 在线链接」或「🌐 网页」
2. 点击「粘贴链接」，在弹出的对话框中输入 URL
3. 点击「开始转换」
4. 结果同样显示在下方文本区域，可保存或复制

## 六、注意事项

1. **转换耗时**：大文件（如几百页的 PDF）可能需要数秒到数十秒，请耐心等待
2. **格式兼容性**：部分格式（如图片描述、音频转文字）可能需要额外安装依赖库，若转换失败会显示具体原因
3. **网络要求**：转换在线链接时需要保持网络连接
4. **中文编码**：所有输出均为 UTF-8 编码，确保目标软件支持 UTF-8 以正确显示中文
5. **.doc 格式**：当前版本不支持旧版 Word（.doc），请先转换为 .docx 格式

## 七、常见问题

**Q：提示"转换失败"怎么办？**

- 检查文件是否损坏或加密
- 确认文件格式是否在支持列表中
- 查看状态栏的错误提示信息

**Q：转换结果为什么是空的？**

- 文件本身可能不包含可提取的文本（如图片型 PDF）
- 尝试使用 OCR 工具预处理后再转换

**Q：转换速度很慢？**

- 大文件正常需要较长时间，请耐心等待
- 在线链接转换速度取决于网络状况
